1. Missing Data
Issue: Missing or null values can distort statistical calculations or break models.

EDA Resolution:

Identify columns with missing values using .isnull().sum() or EDA plots.

Visualize missingness (e.g., heatmaps with seaborn, missingno).

Impute (mean, median, mode) or drop missing data based on insight.

🧪 2. Data Type Mismatches
Issue: Numerical data stored as strings, dates not parsed, categorical values as integers.

EDA Resolution:

Use .dtypes, .info() to inspect types.

Convert with pd.to_numeric(), pd.to_datetime(), astype().

🧼 3. Outliers
Issue: Extreme values skew averages and can mislead models.

EDA Resolution:

Identify via boxplots, histograms, z-score, IQR.

Decide to remove, transform, or cap values.

🌀 4. Duplicated Rows
Issue: Inflates results and introduces bias.

EDA Resolution:

Detect with .duplicated() and remove with .drop_duplicates().

💬 5. Inconsistent Categories / Typos
Issue: Categorical variables may have inconsistent spelling or casing.

EDA Resolution:

Use .unique() or value counts to inspect.

Standardize with .str.lower(), .replace(), .map().

📏 6. Unbalanced Classes
Issue: Classification problems may have skewed class distributions.

EDA Resolution:

Visualize target variable distribution (e.g., countplot).

Apply resampling strategies later if needed (e.g., SMOTE, under-sampling).

📉 7. Skewed Distributions
Issue: Skewed numeric data may need transformation for better modeling.

EDA Resolution:

Use histograms, KDE plots.

Apply log, square root, or Box-Cox transformations.

📊 8. Low Variance / Constant Columns
Issue: Columns with little to no variance offer little information.

EDA Resolution:

Use .nunique() or variance threshold.

Drop if not useful for analysis or prediction.

🔗 9. Multicollinearity
Issue: Highly correlated features may reduce model interpretability.

EDA Resolution:

Correlation matrix, heatmaps.

Drop or combine highly correlated features.

⛔ 10. Invalid or Impossible Values
Issue: E.g., negative ages, percentages over 100.

EDA Resolution:

Use domain knowledge to filter and fix.

Visual or statistical anomaly detection.

Tools Commonly Used in EDA:
Python: Pandas, Seaborn, Matplotlib, Plotly, Sweetviz, Pandas-Profiling, Autoviz.

Power BI / Tableau: For interactive and visual EDA.

EDA Agents (AI): Can automate detection and suggestions using LLMs + rules.
